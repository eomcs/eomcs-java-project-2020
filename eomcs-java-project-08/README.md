# 08 - 기본 문법의 활용

## 학습 목표

- 변수, 상수, 연산자, 조건문, 반복문, 블록, 배열 등의 문법을 활용할 수 있다.

## 구현 목표

- 회원, 프로젝트, 작업 정보를 한 클래스로 합쳐라.
- 사용자로부터 관리 명령어를 입력 받아라.
- 명령어에 따라 작업을 분기하여 처리하라.
