# eomcs-java-project-2020

이 저장소는 온라인 강의 '프로젝트로 배우는, 자바 실전 프로그래밍'의 예제 소스를 보관한다.
이 강의의 목적은 `자바 프로그래밍의 실전 기법`과 `개발 경험`을 전달하는 것이다.
예제 프로젝트를 진행하는 과정 속에서 실전 프로그래밍 기술과 개발 경험을 쌓을 수 있도록 구성하였다.

예제로 진행할 프로젝트는 소규모 팀에서 사용할 간단한 '프로젝트 관리 시스템'을 만드는 것이다.
콘솔 입출력에서 시작하여 웹 애플리케이션까지 단계적으로 진화하도록 구성하였다.
각 단계마다 구현 목표가 있으며, 구현에 필요한 자바 문법과 기법들을 소개한다.

프로젝트의 각 단계를 따라가다 보면 자바 문법을 실전에서 어떻게 활용하는지 그 방법을 배울 수 있다.
유지보수를 좋게 하기 위한 코드 리팩토링 방법이나,
기능 변경이나 확장이 용이한 구조로 애플리케이션을 설계하는 방법도 함께 배울 수 있다.

이렇게 단계적으로 프로젝트를 진행하다보면
과거에서 최근까지 약 20년간 애플리케이션 아키텍처가 변화해 온 과정을 간접적으로 경험할 수 있다.
이를 통해 개발 입문자는 더 빠르게 성장할 수 있을 것이다.
오래전에 구축된 시스템에서 최근에 구축된 시스템까지 유지보수 해야하는 개발자에게도 도움이 될 것이다.

## 대상자

- 자바 기본 문법을 공부중인 분
- 서블릿/JSP를 학습하였거나 학습하려는 분
- C/C++, Python 등 다른 프로그래밍 언어를 알고 있는 데, 자바 프로그래밍을 빠르게 배우고 싶은 분
- 자바 기본 문법을 공부하였는데 어떻게 응용해야 할 지 모르겠는 분
- 다양한 오픈 소스를 자바 애플리케이션에 개발에 적용하는 방법을 배우고 싶은 분
- 스프링 프레임워크 기반 프로젝트에 참여중이거나 참여할 예정인 분
- 자바 웹 애플리케이션 프로젝트의 유지보수를 맡고 있거나 맡을 예정인 분
