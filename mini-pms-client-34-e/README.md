# 34-e. 네트워크 API를 활용하여 C/S 구조로 전환하기 : 다중 클라이언트의 요청 처리

이번 훈련에서는,
- **자바 네트워크 API** 를 사용하여 클라이언트/서버 통신 애플리케이션을 만든다.
